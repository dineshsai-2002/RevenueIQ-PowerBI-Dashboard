# RevenueIQ-PowerBI-Dashboard
Advanced Power BI Sales Dashboard implementing Star Schema modeling, Time Intelligence (CY vs PY), YOY metrics, dynamic slicers, and optimized DAX functions including CALCULATE, SELECTEDVALUE, and COALESCE.


🚀 RevenuePulse – Sales Analytics Dashboard
---------------------------------------------
📌 Overview
-------------
An interactive Sales Dashboard built using Power BI to analyze revenue, profit, and quantity with Current Year vs Previous Year (CY vs PY) comparison and YOY growth insights.
Designed to help businesses monitor performance, identify trends, and make data-driven decisions.

🎯 Key Highlights
📊 CY vs PY Sales & Profit Analysis
📈 Year-over-Year (YOY) Growth Metrics
🌍 Region & State-wise Performance Breakdown
🧠 Optimized DAX Measures
⚙️ Star Schema Data Model
🔄 Blank Handling using COALESCE()

🛠️ Tech Stack
Power BI
DAX (CALCULATE, SELECTEDVALUE, COALESCE)
Data Modeling (Star Schema)
Microsoft Excel

💡 Business Impact
✔ Identified top-performing regions
✔ Tracked growth trends year-over-year
✔ Highlighted performance gaps
✔ Enabled KPI-based decision-making
